# 기본 자료구조 개념 정리



## 정렬

- simple, slow

  - Bubble sort : **O(n^2)**
  - Insertion sort : worst - **O(n^2)** / best : **O(n)**
  - Selection sort : **O(n^2)**

- fast(***resursion***)

  - Quick sort : worst **O(n^2)**, avg **O(n**log**n)**
  - Merge sort : **O(n**log**n)**
  - Heap sort  : **O(n**log**n)**

- O(n)

  - Radix sort
